# 多维对标分析系统 - 快速入门

## 🚀 5分钟快速上手

### 第1步：了解系统结构

我们有4个知识库：
1. **CEO思维模式库**（6位CEO）
   - 位置：`01-knowledge-base/ceo-mindsets/`
   - 用途：对标产品决策逻辑

2. **商业理论库**（10个理论）
   - 位置：`01-knowledge-base/business-theories/`
   - 用途：匹配产品底层逻辑

3. **成功公司案例库**（15个案例）
   - 位置：`01-knowledge-base/company-cases/`
   - 用途：学习成功路径

4. **思维模型库**（10个模型）
   - 位置：`01-knowledge-base/thinking-models/`
   - 用途：主动推理分析

### 第2步：选择一个AI产品试试

例如：**Perplexity**

### 第3步：进行多维对标

#### 3.1 CEO模式对标

1. 打开 `01-knowledge-base/ceo-mindsets/README.md`
2. 看看Perplexity的决策逻辑像哪位CEO
3. 打开对应的CEO文档（如 `elon-musk.md`）
4. 用"识别信号"检查
5. 用"验证问题"深度分析

**示例**：
- Perplexity挑战"搜索=给链接"的假设 → 像Elon Musk的第一性原理思维
- 匹配度：⭐⭐⭐⭐⭐ (90%)

#### 3.2 商业理论对标

1. 打开 `01-knowledge-base/business-theories/README.md`
2. 看看Perplexity符合哪个理论
3. 打开对应的理论文档（如 `disruptive-innovation.md`）
4. 用"匹配方法"检查
5. 提炼可应用原则

**示例**：
- Perplexity从边缘用户切入 → 符合破坏式创新
- 匹配度：⭐⭐⭐⭐ (85%)

#### 3.3 公司案例对标

1. 打开 `01-knowledge-base/company-cases/README.md`
2. 找到类似的成功案例（如 `notion-journey.md`）
3. 对比相似点和差异点
4. 提炼可学习路径

**示例**：
- Perplexity像Notion早期：从个人用户切入，免费策略

### 第4步：主动推理

用5个层次的推理：

1. **Level 1：5 Whys找根因**
   - 工具：`01-knowledge-base/thinking-models/five-whys.md`
   - 时间：15分钟

2. **Level 2：反事实推理**
   - 问题：如果没有AI，会怎样？

3. **Level 3：模式识别**
   - 问题：这个产品符合哪些成功模式？

4. **Level 4：机制设计**
   - 问题：飞轮机制是什么？

5. **Level 5：可证伪设计**
   - 问题：如何验证核心假设？

### 第5步：发散思维

用12个触发器检查：
1. 时间旅行：10年前/后会怎样？
2. 空间平移：不同国家有何不同？
3. 维度升降：最简/最复杂版本？
4. 用户切换：换用户群体会怎样？
5. 商业模式变换：不收费如何运营？
6. 技术替换：没有AI如何实现？
7. 竞品消失：竞品倒闭会怎样？
8. 规模测试：100万用户会崩溃吗？
9. 价值链重构：能否延伸产业链？
10. 失败反推：如何让它失败？
11. 跨界迁移：其他行业有类似模式？
12. 第一性原理：从零开始会怎么设计？

## ❓ 常见问题

### Q1：如果knowledge base里没有我需要的CEO/理论/案例怎么办？

**A**：现在我们只创建了5个核心示例。如果需要其他的：
- Phase 2会补充更多（1周内）
- 或者你可以参考现有模板自己创建

### Q2：我必须完成所有步骤吗？

**A**：不必须。根据需要选择：
- **快速评估**：只做CEO模式对标
- **标准评估**：CEO+理论对标
- **深度评估**：全部流程

### Q3：这个系统适合所有AI产品吗？

**A**：适合大部分，但有些场景可能需要调整。

### Q4：如何使用Prompt文件？

**A**：
1. 打开 `prompts/agent2_deep_analysis_v2.md`
2. 复制Prompt内容
3. 给AI助手（如Claude/ChatGPT）
4. 让它按照流程分析产品
