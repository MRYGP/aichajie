## 🎯 快速开始

### 使用v3.0（推荐流程）

1. **创建Claude Project**
   - 打开 [Claude.ai](https://claude.ai)
   - 创建新Project："三湘问道-AI项目评估"

2. **导入系统指令**
   - 复制 `claude-project-instructions-v3.0.md` 到Custom Instructions
   - 等待加载完成

3. **上传知识库**
   - 上传 `01-knowledge-base/` 下的所有文档
   - 上传 `taxonomy/` 下的YAML文件
   - 等待处理完成

4. **开始评估**
   ```
   帮我评估一个AI项目：
   
   项目名称：[项目名]
   定位：[一句话说明]
   阶段：[idea/MVP/...]
   团队：[X人，背景：...]
   商业模式：[如何赚钱]
   
   请用标准评估模式（推荐）
   ```

5. **查看报告**
   - 报告自动保存到 `02-evaluation-reports/`
   - 使用模板：`templates/deep-evaluation-template.md`

## ⚠️ 重要提示

### v3.0使用注意事项

1. **证据等级标注是强制要求**
   - 所有关键断言必须标注[A/B/C/X]
   - X级证据会自动标红警告

2. **主动推理不可跳过**
   - 至少完成5 Whys + 反事实推理
   - Level 5可证伪设计是强制要求

3. **失败案例必须引用**
   - 每个评估至少1个失败案例
   - 必须标注FM标签
   - 必须对比分析

4. **知识库必须完整**
   - 缺少文档会影响分析质量
   - 定期更新案例库
