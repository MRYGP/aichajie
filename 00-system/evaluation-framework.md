# 三湘问道评估框架总览

## 核心框架

### 7步拆解法
1. 产品基本信息（5分钟）
2. 五维拆解（30分钟）
   - 用户与场景（20%）
   - 产品与价值（20%）
   - 技术与壁垒（20%）
   - 商业模式（20%）
   - 团队与执行（20%）
3. 对标分析（10分钟）
4. 理论映射（10分钟）
5. 导师点评（5分钟）
6. 评分决策（5分钟）
7. 金句提炼（5分钟）

### 11个增值功能
**基础层**：评估打分、风险识别、行动建议、90天验证、完整报告
**启发层**：对标拷问、理念背书
**升级层**：反面案例、时间窗口、可证伪点、学习路径

### 三种模式
- 模式1：快速拆解（<30分钟）
- 模式2：深度拆解（1-2小时）
- 模式3：跨项目对比（2-3小时）

## 评分标准

| 分数 | 级别 | 决策 |
|------|------|------|
| 8-10分 | 优秀 | 强烈跟进 |
| 6-7分 | 良好 | 谨慎观察 |
| 4-5分 | 一般 | 需要改进 |
| 0-3分 | 较差 | 不建议投入 |
