## 🚀 快速开始

### Step 1：创建Claude Project

1. 打开 [Claude.ai](https://claude.ai)
2. 创建新Project："三湘问道-AI项目评估"
3. 复制 `00-system/claude-project-instructions.md` 到Custom Instructions
4. 上传 `01-knowledge-base/` 下的所有文档

### Step 2：评估第一个项目

在Claude Project中输入：
```
帮我评估一个AI项目：

项目名称：[项目名]
定位：[一句话说明]
阶段：[idea/MVP/...]
团队：[X人，背景：...]
商业模式：[如何赚钱]

请用深度拆解模式评估。
```

### Step 3：查看评估报告

- 报告保存在：`02-evaluation-reports/2026-Q1/`
- 使用模板：`02-evaluation-reports/templates/`

## 📊 核心能力

### 7步拆解法
1. 产品基本信息（5分钟）
2. 五维拆解（30分钟）
3. 对标分析（10分钟）
4. 理论映射（10分钟）
5. 导师点评（5分钟）
6. 评分决策（5分钟）
7. 金句提炼（5分钟）

### 11个增值功能
- 基础层：评估打分、风险识别、行动建议、90天验证、完整报告
- 启发层：对标拷问、理念背书
- 升级层：反面案例、时间窗口、可证伪点、学习路径

### 三种模式
- 模式1：快速拆解（<30分钟）
- 模式2：深度拆解（1-2小时）
- 模式3：跨项目对比（2-3小时）

## 💼 使用场景

### 场景1：创业者自查（免费体检）
1. 填写30问问卷（10分钟）
2. AI生成评估报告（1小时）
3. 发现问题、优化方向

### 场景2：投资人筛选（初步尽调）
1. 创业者提交项目信息
2. AI标准化评估（1-2小时）
3. 投资人决定是否深入接触

### 场景3：大赛评审（批量筛选）
1. 参赛者统一填写问卷
2. AI批量生成报告
3. 评委参考报告+人工复核
