# 评分规则说明

## 五维评分（各占20%）

### 1. 用户需求（0-10分）

**评分维度**：
- 痛点真实性（30%）：真伪需求判断
- 使用频率（30%）：高频/低频
- 付费意愿（30%）：已付费/愿意付费/不愿意
- 用户验证（10%）：有多少真实用户

**评分公式**：
```
用户需求分 = 痛点真实性×0.3 + 使用频率×0.3 + 付费意愿×0.3 + 用户验证×0.1
```

---

### 2. 产品价值（0-10分）

**评分维度**：
- 核心功能完整性（40%）
- 差异化程度（30%）
- 用户体验（20%）
- 产品完成度（10%）

**评分公式**：
```
产品价值分 = 功能完整性×0.4 + 差异化×0.3 + 用户体验×0.2 + 完成度×0.1
```

---

### 3. 技术壁垒（0-10分）

**评分维度**：
- 技术难度（40%）：竞品多久能复制
- 数据飞轮（30%）：用户越多越强
- 技术风险控制（20%）
- 技术团队能力（10%）

**评分公式**：
```
技术壁垒分 = 技术难度×0.4 + 数据飞轮×0.3 + 风险控制×0.2 + 团队能力×0.1
```

---

### 4. 商业模式（0-10分）

**评分维度**：
- LTV/CAC比值（40%）
- 变现清晰度（30%）
- 可扩展性（20%）
- 盈亏平衡预期（10%）

**评分公式**：
```
商业模式分 = LTV/CAC×0.4 + 变现清晰度×0.3 + 可扩展性×0.2 + 盈亏平衡×0.1
```

**LTV/CAC评分标准**：
- ≥5：10分
- 3-5：8分
- 1-3：6分
- <1：4分

---

### 5. 团队执行（0-10分）

**评分维度**：
- 创始人匹配度（40%）
- 执行速度（30%）
- 团队完整性（20%）
- 资源获取能力（10%）

**评分公式**：
```
团队执行分 = 创始人匹配×0.4 + 执行速度×0.3 + 团队完整×0.2 + 资源获取×0.1
```

---

## 综合评分

**公式**：
```
总分 = 用户需求×20% + 产品价值×20% + 技术壁垒×20% + 商业模式×20% + 团队执行×20%
```

**决策建议**：
- 8-10分：强烈跟进
- 6-7分：谨慎观察
- 4-5分：需要改进
- 0-3分：不建议投入
