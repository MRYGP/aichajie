## 🎯 决策树速查

### 路径1: 复制/对标
```
□ 破坏式创新潜力 ≥8分? → YES
  └→ 技术壁垒 ≤3分? → YES
      └→ 时机窗口 ≥6个月? → YES
          └→ 团队适配度 ≥3分? → YES
              → ✓ 可以复制!
              
行动: 启动MVP开发 (2-3个月, 预算$15万)
```

### 路径2: 工具使用
```
□ 工具属性ROI ≥5x? → YES
  → ✓ 立即订阅!
  
行动: 
1. 当天订阅
2. 1-2周学习
3. 1个月后评估ROI
4. ROI<5x → 取消
```

### 路径3: 代理/分销
```
□ 客单价 ≥$1000/年? → YES
  └→ 佣金 ≥20%? → YES
      └→ 需要本地化服务? → YES
          → ✓ 值得代理!
          
行动: 
1. 联系创始人
2. 谈判合同 (佣金/区域/独家)
3. 本地化 (汉化/NMPA)
4. 建立销售团队
5. 获取第一个客户 (3-6个月)
```

### 路径4: 观察
```
□ 不符合以上条件
  → 归档到"观察池"
  → 设置下次复查日期 (3/6/12个月)
```

## 📊 评估8维度速查卡

### 维度1: 破坏式创新潜力 ⭐⭐⭐⭐⭐ (10分钟)
```
□ 识别主流产品: [谁?]
□ 识别被忽视人群: [谁?]
□ 功能对比: [少了什么? 用户在意吗?]
□ 定价对比: [便宜多少? ≥50%?]
□ 中国市场: [有类似人群吗? 规模?]

评分:
10分: 完美破坏 (Mirelo级别)
9分: 强破坏 (Cactus级别)
8分: 明确破坏 (Janus级别)
≤7分: 不建议复制
```

### 维度2: 技术壁垒 (5分钟)
```
□ 核心技术: [模型/算法/数据]
□ 依赖: [OpenAI API / 自研]
□ 中国替代: [DeepSeek / 智谱 / 通义]
□ 成本: [$5万 / $20万 / $100万+]

评分:
1分: 极易 (基于API)
2分: 容易 (Fine-tune,数据易得)
3分: 中等 (需要收集数据)
4分: 较难 (自研模型)
5分: 极难 (大规模数据+算力)
```

### 维度3: 时机窗口 (3分钟)
```
□ 美国阶段: [刚融资/有收入/规模化]
□ 中国竞品: [0家/1-2家/3-5家/5家+]
□ 窗口估算: [X个月]

评分:
⭐⭐⭐⭐⭐: 12月+ (立即行动)
⭐⭐⭐⭐: 6-12月 (快速决策)
⭐⭐⭐: 3-6月 (谨慎观察)
⭐⭐: <3月 (已晚)
⭐: 已关闭 (不建议)
```

### 维度4-8: 快速评估 (各2-3分钟)
```
□ 商业模式: [订阅/按次/平台/API] 清晰?
□ 工具ROI: (效率提升*$50/h) / 订阅费 ≥5x?
□ 代理潜力: 客单价≥$1K? 佣金≥20%?
□ 团队适配: 缺什么? [技术/市场/运营/销售]
□ 启动资金: MVP+6月 = $X (<$10万⭐⭐⭐⭐⭐)
```
