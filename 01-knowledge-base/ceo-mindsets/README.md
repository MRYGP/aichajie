# CEO思维模式库

## 简介

本库收录了成功CEO的核心思维模式，用于AI产品分析时的多维对标。通过对比产品决策逻辑与成功CEO的思维模式，可以：

- 识别产品的战略思维来源
- 预测产品的未来发展方向
- 发现潜在的风险和机会
- 提炼可迁移的原则和方法

## 使用方法

### 步骤1：分析产品决策逻辑

观察产品的以下特征：
- 产品定位和差异化策略
- 技术路线选择
- 商业模式设计
- 增长策略
- 团队文化

### 步骤2：对比CEO思维模式

1. 打开本README，浏览6位CEO的核心模式
2. 根据产品特征，初步判断可能匹配的CEO（1-2个）
3. 打开对应的CEO文档，阅读"识别信号"部分
4. 逐一检查产品是否符合这些信号

### 步骤3：深度对标分析

1. 使用"验证问题"部分，对产品进行深度拷问
2. 记录匹配点和不匹配点
3. 提炼可学习点：如果完全按这个CEO的思路，下一步应该做什么？
4. 识别风险点：这个模式在这个场景下的局限是什么？

### 步骤4：输出对标报告

在分析报告中包含：
- **主要对标**：匹配度最高的CEO（1个）
- **次要对标**：部分匹配的CEO（1-2个）
- **匹配度评分**：0-5星
- **可学习点**：具体可执行的动作
- **风险预警**：需要注意的问题

## 示例

**分析Perplexity时**：

1. **初步判断**：产品挑战了"搜索=给链接"的行业假设，符合Elon Musk的第一性原理思维
2. **打开文档**：阅读`elon-musk.md`的"识别信号"部分
3. **验证检查**：
   - ✅ 信号1：挑战行业假设（搜索=直接给答案）
   - ✅ 信号2：技术路线独特（RAG+AI生成）
   - ✅ 信号3：快速迭代（每月更新）
4. **深度分析**：使用"验证问题"部分，发现匹配度90%
5. **输出结论**：主要对标Elon Musk，可学习点：下一步应该做"AI主动提供信息"而非被动搜索
